# Neon Rhythm

A simple, hackable rhythm game built with Vanilla JavaScript.

## Getting Started
Just open `index.html` in your browser! No installation required.

## Controls
- **Start**: Space
- **Play**: D, F, J, K keys

## How to Hack
Check out the `js/` folder to see how it works.
- `song.js`: Change the notes
- `renderer.js`: Change the graphics
- `audio.js`: Change the sounds
